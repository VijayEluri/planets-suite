Jing INSTALLATION:
------------------
1) To install jing on your system, simply extract the contents from the provided zip in 

PSERV/PC/odfvalidator/resources/jing_binaries

to a folder of yuor choice.

2) Afterwards, please set up a system variable JING_HOME and point it to the folder where the jing.jar is located

[somePath]/jing-20091111/bin/

3) Restart your machine (at least, when you are running Windows)
