Detailed WebDAV instructions

From Planets Wiki

Jump to: navigation, search

The instructions below detail how to add a Testbed WebDAV drive to your local machine, allowing you to upload experiment data via your operating system's GUI. Instructions are currently available for Windows XP, Windows Vista and Ubuntu 8.

Adding this drive will require a PLANETS username and password. This is the same username and password used to log in to the PLANETS Testbed.

[edit]

Windows XP

1. Open My Network Places from the Desktop. A link to My Network Places is also available from My Computer under Other Places.

2. Under Network Tasks, click Add a network place. The Add Network Place wizard will open.

3. When asked "Where do you want to create this network place?", select Choose another network location

4. In the Internet or network address box, enter the following address: 

http://testbed.hatii.arts.gla.ac.uk:8080/storage-webdav/repository/default/

5. When prompted, enter your PLANETS username and password. You may check the box the "Remember my password" but bear in mind the security implications: this will allow anyone with access to your computer to view your uploaded experiment files. Checking this box on a public or shared computer is not recommended.

6. Enter a suitable name for the new drive e.g. PLANETS Testbed WebDAV

7. When you click finish, a shortcut to the WebDAV area will now appear in My Network Places

[edit]

Windows Vista

1. Select Computer from the Start menu

2. Click on the Map network drive button

3. Click on the link that says "Connect to a Web site that you can use to store your documents and pictures"

4. If a screen asking you how you connect to the internet appears, click Cancel to use your default internet connection.

5. When asked "Where do you want to create this network location?", click on Choose a custom network location

6. In the Internet or network address box enter the adress below: 

 http://testbed.hatii.arts.gla.ac.uk:8080/storage-webdav/repository/default/

7. When prompted, enter your PLANETS username and password. You may check the box the "Remember my password" but bear in mind the security implications: this will allow anyone with access to your computer to view your uploaded experiment files. Checking this box on a public or shared computer is not recommended.

8. Enter a suitable name for the new drive e.g. PLANETS Testbed WebDAV

9. The new drive will now appear under Computer > Network Location

[edit]

Ubuntu 8

1. From the Places menu, select Connect to Server...

2. In the Service Type drop-down, select WebDAV (HTTP)

3. In the Server box, enter the following address: 

 testbed.hatii.arts.gla.ac.uk:8080/storage-webdav/repository/default/

4. Tick the Add bookmark box and enter a suitable name for the bookmark, e.g. PLANETS Testbed WebDAV

5. Click Connect

6. When prompted, enter your PLANETS username and password. You may choose "Remember forever" to store your password but bear in mind the security implications: this will allow anyone with access to your computer to view your uploaded experiment files. Selecting this option on a public or shared computer is not recommended.

7. The WebDAV drive is now available under Bookmarks in the File Browser
