The Planets Testbed is a web-based software application developed within the EU-funded Integrated Project PLANETS (Preservation and Long term Access through Networked Services, IST-5-033789). Its aim is to provide a controlled environment for the experimentation and evaluation of preservation tools and strategies.

The Planets Testbed is accessible to any person interested in executing experiments on digital long term preservation. To conduct experiments a user must register with the Planets Testbed. A user has to indicate name, institution, valid e-mail address. It is our policy that this information is private and confidential. Accordingly, the personal information you provide to us is stored in a secure location, and is accessible only by designated staff. It will be used only for the purposes for which you provide the information. We will not sell, rent or trade any personal information you provide to us.

The Planets Testbed contains:

  • a software suite developed within the Planets project
  • first-party and third-party web based services, wrapped by the Planets project for the purpose of using them within the Testbed
  • a collection of test files (Corpora) gathered for the special purpose of experimentation within the Planets Testbed

A registered Testbed user can:

  • use the existing workflows created by the Testbed team
  • execute experiments with the tools and services integrated in the test suite
  • use the Corpus and execute experiments on the performance of integrated tools and services on those files
    Note: The provided data is carefully checked on IPR. All material used may be used for assessing services within the Testbed. The Planets Testbed does not assume responsibility for any other use.
  • upload own data to test within Testbed.
    Note: All data uploaded to Testbed is visible to all other Testbed users. The Planets Testbed does not assume responsibility on IPR attached to uploaded data by any user
  • look at and examine all other experiments executed within the Testbed
  • assess as many experiments as wished
  • use the test results for further planning of his/her tasks
  • contact the Testbed Helpdesk to ask for technical and other support
  • access the source code and use it for own developments.

A Testbed user may not

  • download data from the Corpus (see IPR-notes within the Corpus) or other parts of the data registry for use outwith the Planets Testbed.

Testbed users are required to agree to the following terms:

  • Provided data may only be used for internal testing purposes. Any other, especially public or commercial use, has to be agreed on beforehand with the individual data provider.
  • Uploading data to the Planets Testbed is undertaken at the user's own risk.
  • The Planets Testbed does not take responsibility for any digital object uploaded by a user
  • When integrating the Planets Testbed or parts of the Planets Testbed with a new or existing third party application or system Planets shall be informed via the Testbed Helpdesk (helpdesktb@planets-project.eu)
