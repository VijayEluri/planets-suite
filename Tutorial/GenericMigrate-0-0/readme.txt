GenericMigrate-0-0
==================

This package includes a stand-alone service GenericMigrate-0-0 as
well as a client MigrateClient_0 for this service.

The sole prerequisite to run the service and/or client is the
availability of a JAVA instance (JAVA SE 1.5 or higher).

Execute
- publish.cmd from the bin folder in order to run the service.
  (Ctrl-C stops it)
- client.cmd from the bin folder in order to run the client
