Managing the file list

The file list contains the files that have been selected for identification. On startup the list is empty. Files must be added before the identification process can be launched.

To work with the file list see:

  • Adding files to the list
  • Removing files from the list
  • Opening a saved file list
  • Saving the file list
  • Exporting the file list to CSV

Notes

  • The file paths can be displayed/hidden by selecting/unselecting the Show file paths option on the Tools menu.
