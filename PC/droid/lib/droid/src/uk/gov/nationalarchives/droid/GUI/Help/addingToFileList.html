Adding files to the list

To add files to the current list

  1. Either click on the Add Files button or click on Add Files ... in the Edit menu to display the Select Files/Folders dialog.
  2. In the Select Files/Folders dialog, click on the files and/or folders, which you would like to add.
  3. Click on Add Files to add the selected file(s) to the list or Cancel to close the dialog without adding files.

Notes

  • The contents of a folder or drive can be expanded by clicking on the "+" next to the folder name
  • To select multiple files and/or folders hold the Ctrl or Shift key when clicking on files or folders
  • Selecting a folder will add all the files inside the folder
  • Selecting a folder when Include sub-folders is checked will add all files inside the folder and, recursively, any of its subfolders
