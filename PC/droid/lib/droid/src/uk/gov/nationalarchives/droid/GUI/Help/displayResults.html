Displaying results

Once the file identification is complete, the File list pane shows the identification status of each file using the following icons:

  •   a positive identification has been made based on the binary signature of the file
  •   a tentative identification has been made based on the file extension alone because no binary signature information was available for this file format
  •   no hits were found for this file
  •   an error occurred while attempting to identify this file

Selecting an identified file in the File list pane, brings up a list of identified file formats in the Identification results pane. Each file format is described by its Pronom Unique IDentifier (PUID), along with the name, version and MIME type of the format. Its identification status can be:

  • Positive (Specific) if the file satisfies a binary signature which identifies a single file format
  • Positive (Generic) if the file satisfies a binary signature which identifies several file formats
  • Tentative if the file has a file extension used by the file format, and there are no available binary signatures for this file format

Selecting a file with an error status in the File list pane, brings up a description of the error in the Identification results pane.
