Available command line options

DROID can be run in command line mode using the following arguments

  • Display the command line options
          -H
  • Display the DROID application version
          -V
  • Display the signature file version
          -Vxxx      where xxx is the signature file name
  • Check whether the PRONOM website has a newer signature file
          -Cxxx      where xxx is the local signature file
  • Download the latest signature file from PRONOM website
          -Dxxx      where xxx is the name of the file to download to
  • Quiet mode: only print errors to the command line
          -Q
  • Run the analysis on a comma separated list of files
          -Lxxx      where xxx is a comma-separated list of files
          -Syyy      where yyy is the signature file
          -Ozzz      where zzz is the base name for the output files
          -Faaa      where aaa is a comma-separated list of output formats from XML,CSV (defaults to XML only)
  • Run the analysis on the files in a file list file
          -Axxx      where xxx is the file list file
          -Syyy      where yyy is the signature file
          -Ozzz      where zzz is the base name for the output files
          -Faaa      where aaa is a comma-separated list of output formats from XML,CSV (defaults to XML only)
