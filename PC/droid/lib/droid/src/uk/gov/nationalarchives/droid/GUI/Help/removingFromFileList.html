Removing files from the list

To remove one or more files from the File List

  1. Select the file(s) you would like to remove from the list.
  2. Either select Remove Files from the Edit menu or click on the Remove Files button.
  3. A dialog box will apear, asking for confirmation to remove the files. Click Yes to remove selected files , or click No to return to the main window without removing any files.

Notes

  • To select multiple files hold the Ctrl or Shift key when clicking on files in the list.
